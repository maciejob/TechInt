# Lab6 – Notatki

## Wymagania

- **Docker Engine** ≥ 24
- **Docker Compose plugin**
- Przeglądarka internetowa

## Instrukcja uruchomienia

W katalogu Lab6 wykonać:
docker compose up -d --build

Interfejs użytkownika (UI):
http://localhost:8085

Backend API:
http://localhost:8005/api

## Zatrzymanie aplikacji

docker compose down
